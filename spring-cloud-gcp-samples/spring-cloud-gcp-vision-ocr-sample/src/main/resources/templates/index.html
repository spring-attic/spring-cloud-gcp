Process Documents

This application allows you to specify PDF or TIFF documents to upload to your Google Storage bucket and run optical character recognition (OCR) analysis on it.

All operations will be done your Google Storage Bucket specified in application.properties: [[${ocrBucket}]]

Specify a document URL to upload and analyze:

Note: You may specify any PDF document via the following methods:
  • PDFs accessible online through http:// (e.g. http://www.example.com/document.pdf)
  • PDFs in the application resources/ directory through "classpath:". For example, if you want to specify the sample PDF provided with this demo at resources/test_pdfs/test.pdf, you will specify "classpath:test_pdfs/test.pdf".
